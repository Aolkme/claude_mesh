## 设备使用

### Q1: 如何查看设备节点 ID？

**方法一：通过 CLI**
```bash
./meshgate-cli nodes
```

**方法二：通过串口**
连接串口（115200 波特率），设备启动时会输出节点信息。

**方法三：通过 UI**
在 Web 界面的 Nodes 页面查看节点列表。

---

### Q2: 设备无天线通电会有什么后果？

⚠️ **严重警告**：无天线通电可能导致：
- 射频模块永久损坏
- 发射功率过大导致芯片烧毁
- 设备无法正常工作

**务必在通电前安装天线！**

---

### Q3: 如何恢复设备出厂设置？

**操作步骤：**
1. 设备上电状态下
2. 长按按键 10 秒
3. LED 红灯闪烁表示开始重置
4. 等待设备重启完成

**注意**：恢复出厂设置将清除所有配置，包括入网信息。

---

### Q4: 设备 LED 状态代表什么？

| LED 状态 | 含义 |
|----------|------|
| 🔵 蓝色常亮 3 秒 | 未入网状态 |
| 🟢 绿色常亮 3 秒 | 已入网，正常工作 |
| 🟢/🔵 闪烁 | 正在发送信息 |
| 🔴 红色闪烁 | 恢复出厂设置中 |

## 网络配置

### Q5: 如何修改 LoRa 频段？

LoRa 频段由区域设置决定，修改需要：

1. 通过 Admin 命令修改设备区域配置
2. 确保修改后的频段符合当地法规
3. 所有网络节点需要使用相同频段

```bash
# 查询当前配置
./meshgate-cli admin get-config --node !aabbccdd

# 修改区域（需要对应权限）
# 具体命令请参考管理手册
```

---

### Q6: 私有信道有什么作用？

私有信道用于：
- 隔离不同 Mesh 网络
- 加密通信内容
- 防止未授权设备接入

同一网络内的所有设备必须使用相同的信道名称和密钥。

---

### Q7: 如何配置信道密钥？

**方式一：自动生成**
系统自动生成随机密钥，复制到所有节点。

**方式二：自定义**
输入自定义密钥（32字节），确保所有节点一致。

**方式三：通过网关下发**
节点入网后，网关自动下发信道配置。

## 入网问题

### Q8: 节点入网失败怎么办？

**排查步骤：**

1. **确认节点状态**
   - LED 显示蓝色（未入网状态）
   - 节点已上电

2. **确认通信范围**
   - 节点与网关距离是否过远
   - 是否有遮挡物

3. **确认频段一致**
   - 节点和网关使用相同频段

4. **查看网关日志**
   ```bash
   # 查看日志
   tail -f meshgateway.log
   ```

5. **重置后重试**
   - 恢复出厂设置
   - 重新执行入网操作

---

### Q9: 入网后节点离线是什么原因？

**可能原因：**

| 原因 | 解决方法 |
|------|----------|
| 电源耗尽 | 检查/更换电池 |
| 信号中断 | 检查节点位置或增加中继节点 |
| 设备故障 | 联系技术支持 |
| 固件异常 | 尝试远程重启或重置 |

---

### Q10: 如何批量入网多个节点？

**CLI 方式：**
```bash
# 一键入网所有未入网节点
./meshgate-cli enroll-all
```

**UI 方式：**
在节点列表页面，选择「全部入网」按钮。

## 通信问题

### Q11: 消息发送失败怎么办？

**排查步骤：**

1. 确认目标节点在线
2. 确认目标节点已入网
3. 检查信道配置是否一致
4. 检查信号质量（SNR/RSSI）
5. 尝试发送广播消息测试网络

---

### Q12: 通信距离不够怎么办？

**解决方案：**

| 方法 | 说明 |
|------|------|
| 增加中继节点 | 在路径中间部署中继节点 |
| 提高安装高度 | 节点安装在更高位置 |
| 使用高增益天线 | 更换增益更高的天线 |
| 减少遮挡 | 调整位置避开障碍物 |

---

### Q13: 如何查看信号质量？

```bash
# 查看节点详情，包含信号信息
./meshgate-cli node --id !aabbccdd
```

**信号指标：**
| 指标 | 良好 | 一般 | 较差 |
|------|------|------|------|
| SNR | > 0 dB | -5 ~ 0 dB | < -5 dB |
| RSSI | > -90 dBm | -100 ~ -90 dBm | < -100 dBm |

## 电源与续航

### Q14: 节点电池能用多久？

续航时间取决于：
- 唤醒间隔
- 唤醒时长
- 环境温度
- 信号强度
- 电池容量

**参考数据（2000mAh 电池）：**

| 唤醒间隔 | 唤醒时长 | 预估续航 |
|----------|----------|----------|
| 5 分钟 | 2 分钟 | 15-30 天 |
| 15 分钟 | 2 分钟 | 30-60 天 |
| 30 分钟 | 1 分钟 | 60-90 天 |
| 1 小时 | 1 分钟 | 90-180 天 |

---

### Q15: 太阳能供电如何配置？

**建议配置：**
- 太阳能板：5W 以上
- 电池：2000mAh 以上锂电池
- 充电管理：TP4056 或类似模块

**安装建议：**
- 太阳能板朝南（北半球）
- 倾斜角度约等于当地纬度
- 避免阴影遮挡

## 其他问题

### Q16: 网关支持同时连接多少节点？

理论支持数百个节点，实际受以下因素影响：
- 网络带宽
- 消息频率
- 信道数量
- 硬件性能

**建议**：单网关管理节点不超过 100 个。

---

### Q17: 支持 Windows 系统吗？

支持。网关服务可在 Windows 上运行：
- 使用 WSL 或原生 Windows 版本
- 串口驱动需正确安装
- 配置路径使用 Windows 格式

---

### Q18: 如何更换管理员网关？

更换管理员需要：
1. 使用公司私钥生成签名
2. 发送 ChangeAdmin 请求
3. 新网关成为管理员

详细步骤请参考技术文档。

---

### Q19: 设备可以在户外使用吗？

可以，但需要注意：
- 使用防水外壳（防护等级 IP65 以上）
- 注意工作温度范围（-40°C ~ +85°C）
- 考虑雷电防护
- 建议使用太阳能+电池供电
