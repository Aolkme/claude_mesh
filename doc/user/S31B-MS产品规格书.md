## 一、产品概述

S31B-MS 是 Thingseye 推出的 Meshtastic Mesh 网络节点设备，专为物联网数据采集和无线组网通信而开发。

### 1.1 产品特性

- 基于 STM32WLE5 芯片设计
- 支持 LoRa 远距离通信
- 支持 Mesh 自组网
- 低功耗设计
- 开源 Meshtastic 固件

### 1.2 应用场景

- 智慧农业
- 智慧工业
- 资产追踪
- 环境监测
- 应急通信
